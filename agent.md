# AI Agent Rules

1. Node.js Package Manager:
   - **ALWAYS** use `pnpm` for any Node.js package management tasks.
   - If `pnpm` is not installed, download and install it first before running any module commands (`npm install -g pnpm`).
   - Do NOT use `npm` or `yarn` commands (e.g., use `pnpm install`, `pnpm run dev`, `pnpm build`, etc.).
